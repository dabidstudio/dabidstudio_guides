# 텔레그램 봇 토큰 발급 방법

## 1. 텔레그램 실행하기

* 텔레그램 앱 또는 [웹 텔레그램](https://web.telegram.org/)에 로그인합니다.
* 계정이 없다면 먼저 회원가입을 진행합니다.



## 2. BotFather 검색해서 대화 시작하기

* 상단 검색창에 `@BotFather` 또는 'botfather' 입력
* 파란 체크표시가 있는 공식 계정을 클릭합니다.
* 대화창이 열리면 `/start` 명령어를 입력하여 시작합니다.

  <img src="placeholder_step2.png" alt="BotFather 시작화면" width="800">
![image](https://github.com/user-attachments/assets/0cd81852-77c2-46fa-9c57-447241a47aa2)

---

## 3. 새 봇 생성하기 + 이름 정하기

* `/newbot` 입력
* BotFather가 봇 이름을 물어보면 원하는 이름을 입력합니다. (예: `MyTestBot`)

  <img src="placeholder_step3.png" alt="BotFather 봇 생성" width="800">

---

## 4. 사용자 이름(아이디) 정하기

* 사용자 이름은 반드시 `bot`으로 끝나야 합니다.
* 예: `mytest_bot`, `sample123_bot`
* 이미 사용 중인 이름은 사용할 수 없습니다.

  <img src="placeholder_step4.png" alt="봇 사용자 이름 입력" width="800">

---

## 5. 봇 생성 완료 및 토큰, 링크 확인

* 봇 생성이 완료되면 아래와 같은 메시지가 나옵니다.
* `"Use this token to access the HTTP API"` 아래의 긴 문자열이 바로 **API 토큰**입니다.
* 복사해서 안전하게 저장해두세요!

  <img src="placeholder_step5.png" alt="봇 토큰 확인" width="800">
![image](https://github.com/user-attachments/assets/ff27704b-8c44-4606-812d-cee1e056ddb2)

---

## 6. 봇 링크 확인하기

* 생성된 봇은 `https://t.me/내봇아이디` 주소로 접속 가능합니다.
* 예: `https://t.me/mytest_bot`
